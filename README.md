## Project setup

```bash
$ npm install
$ npx prisma migrate dev --name init
$ npx prisma db seed
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
